### TC-01 Создание нового пользователя

**Описание:**
Проверка корректного создания нового пользователя в системе со всеми необходимыми данными.

**Предусловия:**
Приложение запущено. Пользователь не авторизован.

**Последовательность действий:**
1. На стартовом экране ввести n для создания нового профиля.
2. Ввести новый логин: Borob.
3. Ввести пароль: 123.
4. Ввести имя: Crendel.
5. Ввести фамилию: Vicheslav.
6. Ввести год рождения: 1999.

**Ожидаемый результат:**
- В консоли появляется сообщение "Новый профиль успешно создан.".
- Сразу после этого выводится приветственное сообщение: "Добро пожаловать, Пользователь:    Crendel Vicheslav, Возраст: 27 лет!".
- Система переходит в режим ожидания команды от авторизованного пользователя (отображается >).

**Скриншоты:**
- `Screenshots/TC-01.png`
